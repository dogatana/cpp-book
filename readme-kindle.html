Kindle版について

変更内容

Kindle版はオリジナルのものに対して次の変更を加え、Kindleで読める形式へ変換したものである。
  • 表紙画像ファイルの追加
  • readme-kindle.html(本ファイル） をファイル追加
  • README を readme.html としてファイル追加
  • LICENSE を license.html としてファイル追加
  • c++11-Syntax-and-Feature.xhmtl に次の変更を加え、main.xhtml としてファイル追加>
    • script の削除
    • css の body セレクタの定義削除
    • css の h1/h2/h3/h4/h5/h6 セレクタのフォントサイズ指定の変更
    • css の div#content セレクタの line-height 指定追加
  • Kindle用ナビゲーションファイル nav.xhtml の追加
